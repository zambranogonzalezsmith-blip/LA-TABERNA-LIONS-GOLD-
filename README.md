🛡️ LIONS-GOLD | Sistema Mercatus Uniti v2.2
Plataforma Táctica de Intercambio P2P para Albion Online

📋 Descripción del Proyecto
Mercatus Uniti es una infraestructura digital diseñada para el gremio LIONS-GOLD. Su objetivo es centralizar y asegurar las transacciones de Silver, ítems y recursos entre miembros, eliminando el riesgo de estafas y automatizando el cálculo de impuestos (fees) para el crecimiento del clan.

🚀 Características Principales
Radar de Identidad (IGN Verification): Integración con la API oficial de Albion Online para validar la existencia de jugadores en tiempo real.

Calculadora de Tributos: Descuento automático del 0.06% para el fondo de reserva del gremio.

Comando Central (Admin Panel): Panel táctico con métricas de volumen transaccional, recaudación de impuestos y control de reputación.

Protocolo de Exclusión (Blacklist): Sistema de bloqueo inmediato para usuarios detectados en actividades sospechosas o infracciones.

Arquitectura de Datos: Persistencia mediante SQLite con actualización dinámica de "Última Carga".

🛠️ Stack Tecnológico
Frontend: HTML5, Tailwind CSS, FontAwesome 6.

Backend: Node.js, Express.js.

Base de Datos: SQLite3 (Capa de persistencia táctica).

Integraciones: Albion Online GameInfo API.
